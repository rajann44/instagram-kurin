# Clean Feed Launcher

Clean Feed Launcher is a lightweight web app that opens cleaner Instagram feed views in one tap.

## Features

- Open Following feed directly: `https://www.instagram.com/?variant=following`
- Open Favorites feed directly: `https://www.instagram.com/?variant=favorites`
- Minimal, mobile-friendly interface focused on fast access
- Quick share and copy-link actions
- App icons, favicon, and social preview image included

## Notes

- Not affiliated with Instagram or Meta.
- Feed availability, ordering, and behavior are controlled by Instagram.
